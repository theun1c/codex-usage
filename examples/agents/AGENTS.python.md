# AGENTS.md (пример для Python-проекта)

## Scope
Этот файл применяется ко всей директории `backend/`.

## Стек и стиль
- Python 3.12+
- Форматирование: `ruff format`
- Линтинг: `ruff check`
- Типизация: `mypy`
- Тесты: `pytest -q`

## Правила изменений
- Не менять публичные API без явного указания в задаче.
- Для каждого багфикса добавлять регрессионный тест.
- Не добавлять новые зависимости без обоснования в PR.

## Команды проверки перед коммитом
```bash
ruff check .
ruff format --check .
mypy .
pytest -q
```

## Формат итогового отчёта агента
- Краткое summary по изменениям.
- Список запущенных команд и результат.
- Риски/что проверить в ревью.
